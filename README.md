# 🛠️ Ferretería Por Mayor – eCommerce Demo

🌎 Language: [English](#english) | [Español](#espanol)

<a name="english"></a>
## 🇺🇸 English

**Ferretería Por Mayor** is a full stack eCommerce application built with a decoupled architecture between frontend and backend. The frontend communicates with a custom REST API developed using Node.js, Express, and MongoDB.

The platform allows dynamic product management with advanced filtering options and real-time search. It also includes user authentication and online payment integration.

The system is designed with reusable, scalable, and fully responsive components.

---

### 🚀 Features

- Decoupled Frontend & Backend architecture  
- REST API built with Node.js and Express  
- MongoDB database for product management  
- User authentication with Firebase  
- Online payment integration with Stripe  
- Advanced filtering by name, brand, category, and price  
- Real-time product search  
- Persistent shopping cart  
- Fully responsive design  
- Reusable and scalable components  
- Global state management with Zustand

## 🇪🇸 Español

**Ferretería Por Mayor** es una aplicación eCommerce full stack desarrollada con una arquitectura desacoplada entre frontend y backend. El frontend se comunica con una API REST propia construida con Node.js, Express y MongoDB.

La plataforma permite gestionar productos dinámicamente con filtros avanzados y búsqueda en tiempo real. Además, incluye autenticación de usuarios e integración de pagos online.

El sistema está diseñado con componentes reutilizables, escalables y completamente responsive.

---

### 🚀 Funcionalidades

- Arquitectura Frontend y Backend desacoplada  
- API REST con Node.js y Express  
- Base de datos MongoDB para gestión de productos  
- Autenticación de usuarios con Firebase  
- Integración de pagos online con Stripe  
- Filtros avanzados por nombre, marca, categoría y precio  
- Búsqueda de productos en tiempo real  
- Carrito de compras con persistencia  
- Diseño completamente responsive  
- Componentes reutilizables y escalables  
- Gestión de estado global con Zustand
